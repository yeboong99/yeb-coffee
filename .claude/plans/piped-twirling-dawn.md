## 실행 계획

### Phase 1: 빌드 검증

**수행 위치:** `frontend/` 디렉토리

```bash
npm run check-all   # typecheck + lint + format 통과 확인
npm run build       # 빌드 성공 확인
```

- 에러 발생 시 해당 태스크로 돌아가 수정 후 재검증

### Phase 2: 로컬 개발 서버 실행

```bash
cd frontend && npm run dev   # http://localhost:3000
```

---

### Phase 3: Playwright MCP 기반 E2E 테스트

테스트 URL 기준: `http://localhost:3000`

#### 3-1. F022: 리뷰 더보기 검증

1. `http://localhost:3000/capsules/capriccio` 접속
2. 캡슐 상세 페이지 스냅샷 → 리뷰 5개 SSR 렌더링 확인
3. "더보기" 버튼 클릭
4. 스냅샷 → 추가 5개 리뷰 로드 확인 (총 10개)
5. 다시 "더보기" 클릭 → 총 15개 로드 → 버튼 숨김 확인
6. 리뷰 작성 폼: 닉네임, 별점, 내용 입력 → Turnstile CAPTCHA 확인
7. 제출 후 새 리뷰 목록 상단 반영 확인

#### 3-2. F021: 게시글 무한스크롤 검증

1. `http://localhost:3000/community` 접속
2. 스냅샷 → 게시글 10개 SSR 렌더링 확인
3. 페이지 하단으로 스크롤 → 자동 10개 추가 로드 확인
4. 카테고리 탭 전환 → 해당 카테고리 게시글 재조회 확인
5. 모든 게시글 로드 후 완료 메시지 확인

#### 3-3. F020: 홈 페이지 Top 5 랭킹 검증

1. `http://localhost:3000` 접속
2. 스냅샷 → 쿠팡 Top 5 / 서비스 Top 5 섹션 확인
3. 서비스 Top 5 항목 클릭 → 캡슐 상세 페이지 이동 확인
4. 서비스 리뷰 데이터 없을 때 안내 문구 확인

#### 3-4. F001~F012 회귀 테스트 (기존 기능)

| 기능 | URL | 확인 항목 |
|------|-----|-----------|
| 홈 페이지 브랜드 쇼케이스 | `/` | 브랜드 카드 목록 표시 |
| 홈 페이지 인기 게시글 | `/` | 인기 게시글 섹션 표시 |
| 브랜드 목록 | `/brands` | 브랜드 카드 렌더링 |
| 브랜드별 캡슐 목록 | `/brands/nespresso` | 캡슐 카드 목록 |
| 캡슐 검색 | `/brands/nespresso` | 검색어 입력 후 필터링 |
| 강도 필터 | `/brands/nespresso` | 강도 선택 후 필터링 |
| 게시글 작성 | `/community/write` | 폼 렌더링 및 유효성 검사 |
| 게시글 상세 | `/community/{postId}` | 게시글 내용 + 댓글 표시 |
| 댓글 작성 | `/community/{postId}` | 댓글 제출 후 목록 반영 |
