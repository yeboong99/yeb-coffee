# 캡슐 커피 커뮤니티 MVP PRD

## 핵심 정보

**목적**: 네스프레소, 돌체구스토, 일리 등 캡슐 커피 브랜드별 상세 정보를 제공하고, 사용자가 익명으로 캡슐에 대한 리뷰와 경험을 공유할 수 있는 커뮤니티 서비스
**사용자**: 캡슐 커피 머신을 사용 중이거나 구매를 고려 중인 국내 소비자

---

## 사용자 여정

```
1. 홈 페이지 (랜딩)
   ↓ 브랜드 카드 클릭

2. 브랜드 목록 페이지
   ↓ 브랜드 선택

3. 캡슐 목록 페이지 (브랜드별)
   ↓ 캡슐 카드 클릭

4. 캡슐 상세 페이지
   ↓ 사용자 선택 분기

   [리뷰 작성 클릭] → 리뷰 작성 모달 → 제출 → 캡슐 상세 페이지 (리뷰 반영)
   [커뮤니티 글 보기 클릭] → 커뮤니티 목록 페이지 → 게시글 상세 페이지
   ↓

5. 커뮤니티 목록 페이지
   ↓ 글쓰기 버튼 클릭

6. 게시글 작성 페이지
   ↓ 제출 완료

7. 게시글 상세 페이지
   ↓ 댓글 작성 → 같은 페이지에서 댓글 반영
```

## 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|-------------|------------|
| **F001** | 브랜드 카탈로그 조회 | 노션 CMS에서 브랜드 목록 및 브랜드별 캡슐 목록 조회 | 서비스 핵심 콘텐츠 제공 | 홈 페이지, 브랜드 목록 페이지, 캡슐 목록 페이지 |
| **F002** | 캡슐 상세 정보 조회 | 맛, 강도, 향, 호환 머신 등 캡슐 상세 정보 노션 CMS 연동 조회 | 사용자 구매 결정 핵심 정보 | 캡슐 상세 페이지 |
| **F003** | 익명 캡슐 리뷰 작성 | 별점(1-5) + 텍스트 리뷰를 익명으로 작성 및 조회 | 커뮤니티 참여의 핵심 기능 | 캡슐 상세 페이지 |
| **F004** | 커뮤니티 게시글 작성 | 캡슐 관련 경험 및 정보를 익명 게시글로 작성 | 자유 의견 공유를 위한 핵심 기능 | 커뮤니티 목록 페이지, 게시글 작성 페이지 |
| **F005** | 커뮤니티 게시글 조회 | 게시글 목록 및 상세 내용 조회 | 커뮤니티 콘텐츠 소비 핵심 경로 | 커뮤니티 목록 페이지, 게시글 상세 페이지 |
| **F006** | 익명 댓글 작성 | 게시글에 익명 댓글 작성 및 조회 | 커뮤니티 상호작용 최소 단위 | 게시글 상세 페이지 |

### 2. MVP 필수 지원 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|-------------|------------|
| **F010** | 노션 CMS 연동 | 노션 API를 통해 브랜드, 캡슐 데이터 패칭 및 ISR 캐싱 (revalidate: 3600초) | 콘텐츠 관리 인프라 | 홈 페이지, 브랜드 목록 페이지, 캡슐 목록 페이지, 캡슐 상세 페이지 |
| **F011** | 캡슐 검색 | 캡슐명 또는 브랜드명으로 검색 | 콘텐츠 탐색 편의성 필수 | 캡슐 목록 페이지 |
| **F012** | 스팸/어뷰징 방어 | 리뷰·게시글·댓글 작성 시 CAPTCHA 검증 + 서버사이드 IP Rate Limiting | 익명 서비스 무결성 유지 필수 | 캡슐 상세 페이지, 게시글 작성 페이지, 게시글 상세 페이지 |

## 페이지별 상세 기능

### 홈 페이지

> **구현 기능:** `F001`, `F010` | **메뉴 위치:** 헤더 > 홈

| 항목 | 내용 |
|------|------|
| **역할** | 서비스 진입점 및 브랜드 소개 랜딩 페이지 |
| **진입 경로** | 서비스 URL 직접 접속 또는 헤더 홈 메뉴 클릭 |
| **사용자 행동** | 브랜드 카드 목록을 보고 관심 브랜드를 선택해 탐색 시작 |
| **주요 기능** | - 네스프레소, 돌체구스토, 일리 등 브랜드 카드 그리드 표시<br>- 각 브랜드별 캡슐 수, 대표 이미지, 간단 설명 표시<br>- 최신 커뮤니티 인기 게시글 3-5개 미리보기<br>- **브랜드 카드 클릭** 시 캡슐 목록 페이지로 이동 |
| **다음 이동** | 브랜드 카드 클릭 → 캡슐 목록 페이지, 게시글 미리보기 클릭 → 게시글 상세 페이지 |

---

### 브랜드 목록 페이지

> **구현 기능:** `F001`, `F010` | **메뉴 위치:** 헤더 > 브랜드 탐색

| 항목 | 내용 |
|------|------|
| **역할** | 전체 브랜드 목록을 한눈에 보여주는 카탈로그 페이지 |
| **진입 경로** | 헤더 브랜드 탐색 메뉴 클릭 |
| **사용자 행동** | 전체 브랜드를 비교하고 원하는 브랜드를 선택 |
| **주요 기능** | - 노션 CMS에서 가져온 전체 브랜드 목록 표시<br>- 브랜드별 캡슐 종류 수, 호환 머신 유형 표시<br>- 브랜드별 평균 리뷰 점수 표시<br>- **브랜드 선택** 시 해당 브랜드 캡슐 목록 페이지로 이동 |
| **다음 이동** | 브랜드 클릭 → 캡슐 목록 페이지 |

---

### 캡슐 목록 페이지

> **구현 기능:** `F001`, `F010`, `F011` | **메뉴 위치:** 브랜드 선택 후 진입

| 항목 | 내용 |
|------|------|
| **역할** | 특정 브랜드의 전체 캡슐 카탈로그 표시 및 검색 |
| **진입 경로** | 홈 또는 브랜드 목록에서 브랜드 카드 클릭, 또는 검색 결과에서 진입 |
| **사용자 행동** | 브랜드별 캡슐 목록을 탐색하거나 이름으로 검색해 원하는 캡슐 선택 |
| **주요 기능** | - 해당 브랜드의 캡슐 카드 그리드 표시 (이름, 강도, 맛 특성, 썸네일)<br>- 캡슐명 검색 입력창 (F011)<br>- 강도(Intensity) 필터 (약/중/강)<br>- 각 캡슐 카드에 평균 리뷰 별점 표시<br>- **캡슐 카드 클릭** 시 캡슐 상세 페이지로 이동 |
| **다음 이동** | 캡슐 카드 클릭 → 캡슐 상세 페이지 |

---

### 캡슐 상세 페이지

> **구현 기능:** `F002`, `F003`, `F010` | **메뉴 위치:** 캡슐 목록에서 캡슐 클릭

| 항목 | 내용 |
|------|------|
| **역할** | 캡슐 상세 정보 및 익명 리뷰 조회/작성의 핵심 페이지 |
| **진입 경로** | 캡슐 목록 페이지에서 캡슐 카드 클릭 |
| **사용자 행동** | 캡슐 상세 정보(맛, 강도, 향)를 확인하고, 다른 사용자의 리뷰를 읽거나 직접 익명 리뷰를 작성 |
| **주요 기능** | - 노션 CMS 기반 캡슐 상세 정보: 이름, 브랜드, 강도, 맛 설명, 향 특성, 호환 머신<br>- 평균 별점 및 리뷰 수 요약 표시<br>- 익명 리뷰 목록 (별점 + 텍스트, 작성일 표시)<br>- **리뷰 작성** 버튼 클릭 시 리뷰 작성 모달 오픈 (별점 선택 + 텍스트 입력 + 제출)<br>- 커뮤니티에서 이 캡슐 관련 게시글 링크 표시 |
| **다음 이동** | 리뷰 제출 → 같은 페이지 리뷰 목록 갱신, 커뮤니티 링크 클릭 → 커뮤니티 목록 페이지 |

---

### 커뮤니티 목록 페이지

> **구현 기능:** `F004`, `F005` | **메뉴 위치:** 헤더 > 커뮤니티

| 항목 | 내용 |
|------|------|
| **역할** | 사용자 작성 게시글 전체 목록 표시 및 새 글 작성 진입 페이지 |
| **진입 경로** | 헤더 커뮤니티 메뉴 클릭 또는 캡슐 상세 페이지에서 커뮤니티 링크 클릭 |
| **사용자 행동** | 다른 사용자의 게시글을 둘러보거나 새 게시글을 작성 |
| **주요 기능** | - 게시글 목록 표시 (제목, 작성 시간, 댓글 수, 관련 캡슐/브랜드 태그)<br>- 최신순 정렬<br>- 브랜드 태그 필터 (네스프레소, 돌체구스토, 일리 등)<br>- **글쓰기** 버튼 클릭 시 게시글 작성 페이지로 이동 |
| **다음 이동** | 게시글 클릭 → 게시글 상세 페이지, 글쓰기 버튼 → 게시글 작성 페이지 |

---

### 게시글 작성 페이지

> **구현 기능:** `F004` | **메뉴 위치:** 커뮤니티 목록에서 글쓰기 클릭

| 항목 | 내용 |
|------|------|
| **역할** | 익명 커뮤니티 게시글 작성 전용 페이지 |
| **진입 경로** | 커뮤니티 목록 페이지에서 글쓰기 버튼 클릭 |
| **사용자 행동** | 제목과 본문을 입력하고 관련 캡슐/브랜드 태그를 선택 후 익명으로 게시 |
| **주요 기능** | - 제목 입력 필드<br>- 본문 텍스트 에리어<br>- 관련 브랜드 선택 드롭다운 (선택 사항)<br>- 관련 캡슐 선택 드롭다운 (브랜드 선택 시 해당 캡슐 목록 표시, 선택 사항)<br>- **게시하기** 버튼 (익명으로 제출) |
| **다음 이동** | 게시 성공 → 게시글 상세 페이지, 취소 → 커뮤니티 목록 페이지 |

---

### 게시글 상세 페이지

> **구현 기능:** `F005`, `F006` | **메뉴 위치:** 커뮤니티 목록에서 게시글 클릭

| 항목 | 내용 |
|------|------|
| **역할** | 개별 게시글 내용 및 익명 댓글 조회/작성 페이지 |
| **진입 경로** | 커뮤니티 목록 페이지에서 게시글 클릭 또는 홈 게시글 미리보기 클릭 |
| **사용자 행동** | 게시글 본문을 읽고 댓글로 의견을 남기거나 관련 캡슐 정보로 이동 |
| **주요 기능** | - 게시글 제목, 본문, 작성 시간 표시<br>- 관련 캡슐/브랜드 태그 표시 및 클릭 시 캡슐 상세 페이지로 이동<br>- 댓글 목록 표시 (텍스트, 작성 시간)<br>- **댓글 작성** 입력창 + 익명 제출 버튼<br>- 댓글 제출 후 목록 즉시 갱신 |
| **다음 이동** | 캡슐 태그 클릭 → 캡슐 상세 페이지, 브랜드 태그 클릭 → 캡슐 목록 페이지 |

## 데이터 모델

### Brand (브랜드 - 노션 CMS 읽기 전용)

> 노션 데이터베이스에서 관리. API로만 조회하며 DB에 저장하지 않음.

| origin_country | 브랜드 원산지 | Select |
| website_url | 공식 사이트 URL | URL |
| slug | URL용 식별자 (예: nespresso) | Rich Text |

### Capsule (캡슐 - 노션 CMS 읽기 전용)

> 노션 데이터베이스에서 관리. API로만 조회하며 DB에 저장하지 않음.

| 필드 | 설명 | 노션 속성 타입 |
|------|------|--------------|
| id | 노션 페이지 ID | Page ID |
| name | 캡슐명 (예: Ristretto, Volluto) | Title |
| brand_id | 브랜드 노션 페이지 참조 | Relation → Brand |
| intensity | 강도 (1-13 또는 브랜드별 스케일) | Number |
| flavor_notes | 맛 특성 (예: 과일향, 카카오, 견과류) | Multi-select |
| aroma | 향 설명 | Rich Text |
| body | 바디감 설명 (가볍다 / 중간 / 진하다) | Select |
| acidity | 산미 수준 (낮음 / 중간 / 높음) | Select |
| bitterness | 쓴맛 수준 (낮음 / 중간 / 높음) | Select |
| roast_level | 로스팅 정도 (라이트 / 미디엄 / 다크) | Select |
| compatible_machines | 호환 머신 (예: Original Line, Vertuo) | Multi-select |
| image_url | 캡슐 대표 이미지 (외부 호스팅 URL - 만료 없음) | URL |
| description | 캡슐 상세 설명 | Rich Text |
| is_limited | 한정판 여부 | Checkbox |
| slug | URL용 식별자 | Rich Text |

## 기술 스택 (최신 버전)

### 프론트엔드 프레임워크

- **Next.js 16** (App Router) - React 풀스택 프레임워크, ISR을 통한 노션 CMS 캐싱
- **TypeScript 5.6+** - 타입 안전성 보장
- **React 19** - UI 라이브러리

### 스타일링 & UI

- **TailwindCSS v4** (설정파일 없는 새로운 엔진) - 유틸리티 CSS 프레임워크
- **shadcn/ui** - 고품질 React 컴포넌트 (Card, Dialog, Badge, Textarea 등)
- **Lucide React** - 아이콘 라이브러리

### 폼 & 검증

- **React Hook Form 7.x** - 리뷰/게시글/댓글 폼 상태 관리
- **Zod** - 입력값 스키마 검증

### 스팸/어뷰징 방어

- **Cloudflare Turnstile** - 무료 CAPTCHA (리뷰·게시글·댓글 작성 시 서버사이드 토큰 검증)
- **Route Handler IP Rate Limiting** - IP당 시간당 쓰기 요청 횟수 제한 (Upstash Ratelimit 또는 자체 구현)

### 데이터 패칭

- **React Query v5** - 서버 상태 관리 및 캐싱 (노션 API 응답 캐싱)

### 백엔드 & 데이터베이스

- **Supabase** - 익명 리뷰, 게시글, 댓글 데이터 저장 (PostgreSQL)
- **Next.js API Routes (App Router Route Handlers)** - 노션 API 프록시, Supabase 데이터 CRUD

### 패키지 관리 & 배포

- **pnpm** - 패키지 관리
